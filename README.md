# HPT Mockup Previews

Public preview host for HPT mockup HTML files, viewable via **raw.githack.com** so mockups render inline on mobile browsers.

> Files here are illustrative mockups, **not** production code. See the private `houstonposttension/wip-processor` repo for the real spec and implementation.

## How to view a mockup

Prefix any file path with `https://raw.githack.com/houstonposttension/hpt-mockup-previews/main/`.
