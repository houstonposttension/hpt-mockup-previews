# HPT Mockup Previews

Public preview host for HPT mockup HTML files, viewable via **rawcdn.githack.com** so mockups render inline on mobile browsers.

> Files here are illustrative mockups, **not** production code. See the private `houstonposttension/wip-processor` repo for the real spec and implementation.

## How to view a mockup

Prefix any file path with `https://rawcdn.githack.com/houstonposttension/hpt-mockup-previews/main/`.

> Use the `rawcdn.githack.com` host (production CDN), **not** `raw.githack.com` — the latter is the dev URL and serves an intermediate loading/disclaimer page. Both resolve to the same file.
